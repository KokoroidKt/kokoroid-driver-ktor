HttpResponse

data class HttpResponse(val statusCode: Int, val reasonPhrase: String, val json: JsonElement?, val headers: Map<String, List<String>> = emptyMap(), val bodyText: String? = null)

表示一次 HTTP 请求的响应结果，主要面向 JSON API 场景。

这个对象同时保留了解析后的 JSON 数据与原始文本内容，方便在以下场景使用：

  • 响应体是 JSON 时，优先读取 json

  • 响应体不是 JSON，或解析失败时，可通过 bodyText 获取原始文本

  • 需要根据 HTTP 状态码判断请求是否成功时，可使用 isSuccess
